# About detector-api-extension
- The extension detects all JSON APIs called when a website loads, and lets you copy each request as a CURL command.
- The extension is developed by Tuan Nguyen.
- Contact me: nguyenvantuan2391996@gmail.com

## Features
- Detects JSON API requests fired while a page loads.
- Copy any detected request as a ready-to-use CURL command.
- Status badges (success / error) for each request.
- Preserve log toggle, to keep the list across page reloads (similar to DevTools Network tab).
- Live counter of how many requests were detected.
